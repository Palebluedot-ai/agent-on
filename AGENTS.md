# AGENTS.md

## 项目一句话

这是一个把“AI 协作开发项目如何启动”产品化的仓库。

## 当前目标

沉淀一套稳定可复用的方法，让任何新项目都能快速生成：

- 项目规则
- 需求澄清包
- PRD
- milestone
- 开发拆解清单
- skill 骨架

## 文档纪律

1. 每份文档必须尽量自包含。
2. 不依赖“先读另一份文档”才能理解。
3. 可以重复关键事实，不为了去重而交叉引用。
4. 未确认事项必须单列，不能伪装成已确认事实。
5. 每份文档开头都说明自己的职责边界。
6. 先固化规则，再抽象成模板，再考虑自动化。

## 工具偏好

- 文档优先
- 规则优先
- 先小后大
- 不先做平台

## 当前阶段

当前处于：

- 方法论沉淀
- 模板抽象
- skill 拆解设计

尚未进入：

- 完整自动化平台实现

## 不做的事

- 不把这个项目做成一个一开始就很重的平台
- 不为了“通用”牺牲真实项目里的有效做法
- 不接受文档到处跳转引用
