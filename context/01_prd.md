## 1. 产品概述

`project-kickoff-os` 是一个帮助用户把“新项目启动阶段”标准化的系统。

第一版不做复杂产品界面，先把最核心的文档驱动流程固化下来。

## 2. 第一版用户

- 经常和 AI 一起做项目的人
- 需要反复启动新项目的人
- 希望把沟通方式、文档方式、milestone 方式沉淀为 SOP 的人

## 3. 第一版目标

第一版要做到：

1. 明确项目启动阶段有哪些固定产物
2. 形成文档自包含规则
3. 形成需求澄清模板
4. 形成 PRD 模板
5. 形成 milestone 和开发拆解模板
6. 明确后续该拆成哪些 skill

## 4. 第一版产物

- 项目规则文件
- 文档规则文件
- 需求澄清模板
- PRD 模板
- milestone 模板
- skill map

## 5. 第一版不做

- 完整 Web 平台
- 自动生成一切的 orchestration 系统
- 多人协作管理后台
- 复杂数据库设计
